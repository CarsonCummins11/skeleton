Tools like V0, Lovable, Firebase Studio, etc have brought down webapp development time from weeks to minutes. Simultaneously, they have allowed increasingly non-technical users to create webapps, to the point where there exist hundreds of webapps with thousands of paying users with no technical team involved at all. Soon every user will want to generate their own apps to solve their own problems in seconds. 

Leaflet is that tool. Leaflet allows users to generate a webapp to solve exactly their current problem with the same simplicity and ease of use as finding a website on Google. These apps all have access to a shared database of information, so as users generate and use their webapps they all contribute to building up a shared database of knowledge - the modern web, without walled gardens.
